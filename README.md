# iBook

Electron 电子书阅读器，支持 TXT 格式（PDF 即将支持）。

## 功能

- **书架**：添加/拖拽导入、去重提示、删除、200×300 封面卡片
- **翻书过渡**：点击书籍 3D 翻书动画进入阅读
- **阅读器**：四主题、字号调节、翻页动画、自动翻页
- **缩略图导航**：鼠标移至左边缘 20px 显示 200px 侧边栏
- **批注**：彩笔高亮、悬停冒泡显示批注（JSONL 持久化）
- **设置**：背景色/背景图、字体、自动翻页间隔

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
